# GameVault - Catálogo de Videojuegos

## Descripción
GameVault es una aplicación web desarrollada en ASP.NET Core MVC para la gestión de una colección de videojuegos. El proyecto permite visualizar un catálogo dinámico, filtrar títulos por género y registrar nuevos elementos con validaciones de datos.

## Tecnologías Usadas
* **Lenguaje:** C#
* **Framework:** .NET Core MVC
* **Interfaz:** HTML5, CSS3 y Bootstrap 5
* **Iconos:** Bootstrap Icons

## Funcionalidades
* **Catálogo Visual:** Visualización mediante tarjetas (Cards) interactivas.
* **Filtrado dinámico:** Búsqueda por géneros (Metroidvania, RPG, etc.).
* **Gestión de Datos:** Formulario para agregar nuevos juegos con ID autoincremental.
